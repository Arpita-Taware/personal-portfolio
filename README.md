# Personal Portfolio Website 💼

This is a full-stack personal portfolio website built using Node.js, Express, and MongoDB.  
It displays projects and skills dynamically and stores contact form messages in the database.

---

## 🚀 Features

- Responsive modern UI
- Projects & Skills fetched from MongoDB
- Contact form saves messages to database
- REST API integration
- MVC architecture

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Tools: Git, Nodemon, Dotenv

---

## ⚙ How to Run

1. Clone the repository
2. Install dependencies:

```
npm install
```
3.Create a .env file:
```
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

4.Start the server:
```
npm run dev
```
5.Open in browser:
```
http://localhost:5000
```
